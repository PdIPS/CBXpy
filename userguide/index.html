Basic Parameters and nomenclature#

We collect some basic parameters and terminology that are used in the following

symbol

description

\(x\)

The ensemble of points

\(f\)

The objective function

\(M\)

The number of runs

\(N\)

The number of particles

\(d\)

The dimension

\(dt\)

The time step parameter

\(t\)

The current time

\(\alpha\)

The weighting parameter for the consensus point

\(\sigma\)

The noise scaling parameter
